# travels-companions

## Назначение
Портал для поиска компании по интересам с возможностью взаимодействия пользователей
<br>Пример портала: <a href="https://travels-companions.info/help">travels-companions.info</a>
## Что используется
* Бэкенд: laravel
* Фронтенд: SPA на vue.js и bootstrap
* Фронт взаимодействует с бэком через API и WebSocket (laravel-echo, socket.io)
* Портал не является на 100% готовым решением, есть недоработанные моменты.
## Основные моменты
### Пользователи
* Регистрация с подтверждением по смс (повторный запрос кода возможен через десять минут)
* При восстановлении пароля пользователю отправляется СМС с новым паролем (повторный запрос пароля возможен через десять минут)
### События
* Добавление/редактирование изображений с выбором уровня доступа (участники, зарегистрированные пользователи, все посетители) доступно после создания события
* Управление участниками события
* Уведомления 
* За состоявшееся событие автору и участникам летит плюс в карму, за не состоявшееся автор получает минус
### Уведомления по событиям 
* Доставляются через WebSocket
* Отправляются в случае:
  - Изменения события
  - Приглашения на участие в событии
  - Ответа на приглашение/заявку
### Сообщения
* Доставляются через WebSocket
* Есть два типа сообщений: 
  - Личные сообщения
  - Групповые чаты событий. 
 * Групповые чаты событий создаются автоматически при создании события. 
 * Пользователь видит только сообщения написанные после присоединения к событию. 
 * Если пользователь был исключен из участников события, он теряет доступ к чату события.
